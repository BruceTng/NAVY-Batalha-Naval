NAVY - Batalha Naval
| Projeto pessoal desenvolvido em Python com foco em praticar lógica de programação.


Como jogar:

Execute o arquivo Navy.py no terminal

Escolha uma opção no menu

Posicione seus 5 barcos informando as coordenadas X e Y

Tente afundar os navios inimigos antes que suas tentativas acabem

Requisitos:
Python 3.x

Observação:
Desenvolvido no PyCharm — podem ocorrer problemas de compatibilidade ao rodar em outros ambientes. 
Projeto em desenvolvimento — algumas funcionalidades ainda estão sendo implementadas.
